# MONAPY
monachat client with python
## MONAPYについて
+ もなちゃと用クライアントです
+ 使用言語：python3.7
+ 予定：いずれbotを組み込むはず…。
+ 注意：安全性は保証できません、バグが起こるかもしれないで使っていただける方は注意を

## 使用方法
+ python3.7をインストール
+ monapy.pyを実行
+ 一番下にあるエントリーボックスにコマンドを打ち込んでください

## コマンド
+ /connect  
 もなちゃとに接続します(入口に入る)
+ /enter /(ルーム番号)  
 ルーム番号に入ります
+ /enter(space) ←(space)は半角スペース
 入口に入ります
+ /cmt

## 表示
+ room info
